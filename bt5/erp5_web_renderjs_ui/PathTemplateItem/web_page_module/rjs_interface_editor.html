Editor Interface

A rich editor gadget which handle many kind of content type

render
Render a rich editor (ckeditor, codemirror, svgeditor, ...)
option_dict
  • value: text content to view/edit
  • editor: specific editor value (fck_editor, pdf, text_area, svg_editor, onlyoffice, codemirror)
  • content_type: content type of the document
  • maximize: show a maximize button to set the editor in full screen
  • portal_type: portal type of the document
  • editable: is the content modifiable
  • key: the key used to return the new value with getContent
